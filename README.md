# README ainda em desenvolvimento. Essa é a versão inicial, feita apenas para auxiliar a equipe com a execução do esqueleto inicial do projeto Flask.

Instruções iniciais para rodar o projeto:

1. Criar e ativar o ambiente virtual (venv):

python -m venv venv

2. Ativar o Ambiente Virtual:

No Windows (CMD ou PowerShell), use o comando:
venv\Scripts\activate

No Linux ou macOS, use o comando:
source venv/bin/activate

3. Instalar as dependências:

pip install -r requirements.txt

4. Iniciar o servidor:

python app.py
